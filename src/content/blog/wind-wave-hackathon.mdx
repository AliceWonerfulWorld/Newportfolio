## はじめに

「音楽を、魔法にする」──それが72時間のテーマだった。

Wind Wave 2024に参加した。出場チームは全国から約50チーム。
制限時間はきっかり72時間。寝ても覚めても「なんか面白いものが作れないか」を
考え続けた3日間を、ここに記録する。

## アイデアの核心

ブレインストーミングの最初の30分で「杖を振ると音楽が生まれる」というコンセプトが
浮かんだ。技術的な新しさより、**体験としての驚き**を最優先にした判断だ。

チームで即座に決めたこと：

- 専門知識ゼロでも遊べること
- 動きに合わせてリアルタイムで変化すること
- 5秒見ただけで「やってみたい」と思わせること

## 実装の核心：Web Audio API

```js
const audioCtx = new AudioContext();
const oscillator = audioCtx.createOscillator();
const gainNode = audioCtx.createGain();

// マウスの速度を音程に変換
oscillator.frequency.setValueAtTime(
  mapSpeed(velocity, 100, 800),
  audioCtx.currentTime
);
```

マウスの移動速度を周波数にマッピングする部分が最もハマった箇所だ。
単純なリニアマッピングでは「雑音」にしかならない。
スケール（音階）に沿った離散値へのスナッピングが鍵だった。

## 結果と学び

🥉 **3位入賞** + ✨ **特別賞（独創性）** 受賞

「技術の複雑さ」ではなく「体験の驚き」で勝負した判断は正しかった。
次回はモバイルのジャイロセンサーを使ってさらに直感的な操作を目指したい。
