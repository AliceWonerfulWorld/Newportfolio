## 魔法少女変身 WebAR

スマートフォンのブラウザからアクセスするだけで、カメラ越しに魔法少女へ変身できるWebARアプリです。

### 体験の流れ

1. スマートフォンでURLにアクセス
2. カメラを自分に向ける
3. 変身ポーズを決める → AIがポーズを検出
4. エフェクトとともに魔法少女コスチュームが重なる

### 技術スタック

- **AR描画**: Three.js + WebXR API
- **ポーズ検出**: MediaPipe Pose
- **フロントエンド**: React
- **エフェクト**: GSAP + CSS Animations

### ハッカソンでの学び

WebARのリアルタイム処理とモバイルのGPU制約の兼ね合いが最大の挑戦でした。
60fps維持のためにシェーダーの最適化に注力しました。
