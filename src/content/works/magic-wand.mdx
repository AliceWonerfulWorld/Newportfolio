## Magic Wand

Wind Waveハッカソンにて制作した作品。魔法の杖を振るように音楽を生み出すインタラクティブなWebアプリです。

### 概要

マウスやタッチの動きをリアルタイムで解析し、動きのパターンからAIが音楽を生成します。
振り方の強さ・速さ・方向で音色が変化し、誰でも即興演奏を楽しめる体験を実現しました。

### 技術スタック

- **フロントエンド**: React + TypeScript
- **音声処理**: Web Audio API
- **AI生成**: TensorFlow.js (モーション解析)
- **アニメーション**: GSAP

### ハッカソン結果

- 🥉 Wind Wave 2024 — **3位**
- ✨ **特別賞** 受賞
